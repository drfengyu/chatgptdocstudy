快速入门​

G:\PythonProject\OpenAiApiStudy\Openaistudy2

1. 创建虚拟环境​

python -m venv openai-env

2. 激活​

openai-env\Scripts\activate

3. 安装openai库​

pip install --upgrade openai

4. 添加.env文件​

# Once you add your API key below, make sure to not share it with anyone! The API key should remain private.
OPENAI_API_KEY=abc123

5. 导入API KEY​

from openai import OpenAI

client = OpenAI()
# defaults to getting the key using os.environ.get("OPENAI_API_KEY")
# if you saved the key under a different environment variable name, you can do something like:
# client = OpenAI(
#   api_key=os.environ.get("CUSTOM_ENV_NAME"),
# )

6. 激活环境变量​

from dotenv import load_dotenv
load_dotenv()  # 这将加载.env文件中的环境变量

如果您还没有使用 python-dotenv 库，可以通过以下方式安装它：

pip install python-dotenv

8. 通过Discord webhook 发送消息​

如果您需要通过 Discord webhook 发送消息，需要按照 Discord API 提供的 Execute Webhook 端点的要求来构造 HTTP POST 请求。以下是发送消息的基本格式： HTTP POST 请求:

POST https://discord.com/api/webhooks/{webhook.id}/{webhook.token}

请求头部:

Content-Type: application/json

JSON 负载:

{
    "content": "你好，这是一条消息！",
    "username": "Webhook机器人",
    "avatar_url": "https://example.com/image.png",
    "embeds": [
        {
            "title": "嵌入标题",
            "description": "这是一个嵌入内容",
            "color": 5814783
            // 这里可以包含更多嵌入属性
        }
    ],
    // 根据文档规定，您可以在这里包含更多字段
}

在这个 JSON 负载中：

  • "content" 是消息文本。
  • "username" 用来覆盖默认的 webhook 用户名。
  • "avatar_url" 用来覆盖默认的 webhook 头像。
  • "embeds" 是包含嵌入对象的数组。每个嵌入对象包含了定义消息在 Discord 中显示方式的元信息，比如标题、描述、颜色等。 不同属性在 JSON 负载中的使用方式是在 Discord API 文档 中有详细定义的。确保按照您的消息需求来格式化您的 JSON 负载，并遵守 Discord 设置的频率限制，避免在发送消息时出现问题。 要发送 HTTP 请求，您可以使用 curl 工具在终端里操作，或者在 Python 中使用 requests 库来编程实现发送请求。始终确保 webhook URL 包含创建 webhook 时 Discord 提供的正确 webhook.id 和 webhook.token。

9. 使用pyinstaller打包python项目​

在Python中，您可以使用工具比如PyInstaller或者cx_Freeze来将Python项目打包成独立的可 以下是一个简单的步骤来使用PyInstaller来打包Python项目成exe文件：

  • 首先安装PyInstaller:
pip install pyinstaller
  • 在命令行中进入您的项目文件夹，然后运行以下命令:
pyinstaller --onefile your_script.py
这里`your_script.py`是您的Python项目的入口文件名。PyInstaller会创建一个build文件夹
